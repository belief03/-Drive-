# あおいろサポート — 青色申告 帳簿ツール

個人事業主向けの青色申告用簡易帳簿Webアプリです。  
取引タイプで選んで入力、勘定科目のリード型提案、P/L・B/S、CSVエクスポートに対応しています。

## 必要な環境

- Python 3.10 以上
- （推奨）仮想環境: `python -m venv venv` のあと `venv\Scripts\activate`（Windows）

## セットアップと起動

```bash
pip install -r requirements.txt
python app.py
```

ブラウザで http://127.0.0.1:5000/ を開きます。  
初回は業種を選択する初期設定画面が表示されます。

## 主な機能

- **初期設定**：業種を選択（設定画面でいつでも変更可）
- **取引登録**：売上・雑収入・経費・振替・借入・返済・事業主貸借・その他
- **経費のリード型提案**：取引先を入力すると、過去の同じ取引先の勘定科目を提案
- **前年整合性アラート**：経費で別の科目を選んだ場合、同一取引先の過去科目を案内
- **取引一覧**：登録した取引の一覧表示
- **P/L・B/S**：年間の損益計算書・貸借対照表
- **CSVエクスポート**：指定年の取引一覧をCSVでダウンロード（確定申告用の土台）
- **設定**：業種の変更、固定費の自動反映のON/OFF
- **固定費の自動反映**：設定でONにすると、固定費ページを開いたときに今月分が未反映なら自動で反映。または `python scripts/apply_recurring.py` を毎月1日などにタスクスケジューラで実行しても可

## データの保存場所

- SQLite データベース: プロジェクトフォルダ内の `instance/account_book.db`
- バックアップは `instance` フォルダごとコピーするか、CSVエクスポートを定期保存することを推奨（帳簿・領収書は7年保存が目安です）
