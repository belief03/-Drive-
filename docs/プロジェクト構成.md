# プロジェクト構成（青色申告 帳簿ツール）

```
account book/
├── app.py                 # Flask エントリポイント
├── config.py              # 設定（DB・環境変数・Google Drive 等）
├── models.py              # DB モデル（Account, Transaction, Setting, Industry, RecurringExpense）
├── database.py            # DB 初期化・シード・マイグレーション
├── services.py            # 勘定科目提案・同一取引先チェック
├── drive_service.py       # Google Drive API（OAuth・アップロード）
├── requirements.txt
├── run.bat                # Windows 用起動スクリプト
├── README.md
│
├── routes/                # ルート（ブループリント）
│   ├── __init__.py
│   ├── main.py            # トップ・初期設定（業種選択）
│   ├── transactions.py    # 取引一覧・登録・編集・領収書画像・払い済み
│   ├── settings_bp.py     # 設定（業種・固定費自動反映・Google Drive 連携）
│   ├── reports.py         # P/L・B/S・CSV エクスポート・領収書まとめ・PDF
│   ├── hints.py           # 科目のヒント一覧
│   ├── knowledge.py       # 知識・豆知識
│   ├── recurring.py       # 固定費（登録・反映・自動反映）
│   └── csv_import.py      # CSV 取込（アップロード・プレビュー・一括登録）
│
├── templates/             # Jinja2 テンプレート
│   ├── base.html
│   ├── setup.html
│   ├── transactions/      # list, new, edit
│   ├── settings/          # index
│   ├── reports/           # pl_bs, receipts
│   ├── hints/             # index
│   ├── knowledge/         # index
│   ├── recurring/         # index, form, apply
│   └── csv_import/        # upload, preview
│
├── data/                  # マスタ・静的データ
│   ├── hints.json
│   └── knowledge.json
│
├── docs/                  # 設計・手順
│   ├── 勘定科目と入力項目.md
│   ├── 壁打ちと実装メモ.md
│   ├── Googleドライブ連携.md
│   ├── 税理士相談ポイント.md  # 税理士に相談すべき点のまとめ
│   └── プロジェクト構成.md（本ファイル）
│
├── scripts/
│   └── apply_recurring.py # 固定費を反映する CLI（cron/タスクスケジューラ用）
│
└── instance/              # 実行時に生成（git では無視推奨）
    ├── account_book.db    # SQLite DB
    └── receipts/          # 領収書画像
```

## 主な URL プレフィックス

| プレフィックス | 役割 |
|----------------|------|
| `/` | トップ・初期設定 |
| `/transactions` | 取引一覧・登録・編集・領収書 |
| `/transactions/import` | CSV 取込 |
| `/settings` | 設定・Google Drive 連携 |
| `/reports` | P/L・B/S・領収書まとめ・CSV エクスポート |
| `/hints` | 科目のヒント |
| `/knowledge` | 知識・豆知識 |
| `/recurring` | 固定費 |
